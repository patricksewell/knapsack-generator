Dual-Budget Knapsack Generator

This variant generates one set of items and solves the 0/1 knapsack problem under two different budgets (low and high). This is useful for studying how the same set of items leads to different optimal solutions when the budget constraint changes.

All item generation settings work identically to the main generator.
